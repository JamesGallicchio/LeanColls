        inductive Lean.Server.FileWorker.FileSetupResultKind :
        Type

        Categorizes possible outcomes of running lake setup-file.

        • success: Lean.Server.FileWorker.FileSetupResultKind

          File configuration loaded and dependencies updated successfully.

        • noLakefile: Lean.Server.FileWorker.FileSetupResultKind

          No Lake project found, no setup was done.

        • importsOutOfDate: Lean.Server.FileWorker.FileSetupResultKind

          Imports must be rebuilt but --no-build was specified.

        • error: String → Lean.Server.FileWorker.FileSetupResultKind

          Other error during Lake invocation.
