      def Mathlib.Tactic.rename' :
      Lean.ParserDescr

      rename' h => hnew renames the hypothesis named h to hnew. To rename several hypothesis, use rename' h₁ => h₁new, h₂ => h₂new. You can use rename' a => b, b => a to swap two variables.
