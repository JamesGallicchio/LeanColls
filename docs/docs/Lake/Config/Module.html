    structure Lake.Module :
    Type

    A buildable Lean module of a LeanLib.

    • lib : Lake.LeanLib
    • name : Lake.Name
    • keyName : Lake.Name

      The name of the module as a key. Used to create private modules (e.g., executable roots).
