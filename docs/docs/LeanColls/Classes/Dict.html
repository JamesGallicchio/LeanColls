    Dictionaries #

    Dictionaries are collections which encode relational data. The math model is a partial function α → Option β.

    source
    structure LeanColls.Dict.KeySet (C : Type u) :
    Type u

    Represents the key set of a C dictionary.

    This is a trivial wrapper structure, on which dictionary implementations should provide a Bag instance.
